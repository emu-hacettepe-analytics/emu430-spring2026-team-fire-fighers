Title Of The Project : "Analysis on Forest Fires in Turkey"

Description About the Project: In this project, we gathered various data from Orman Genel Müdürlüğü(OGM) to analyze number of fires and area effected by the fires
in Turkey by province. We also tried to analyze the main reasons behind the fires. We also did projection studies  and analysis for forested areas
for the country.
